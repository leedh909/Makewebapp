*(1-1)인터넷 -ip와 도메인
모든사람이 접근할 수 있도록 하기 위해서 자가서버 운영한다.
IP 주소(Internet Protocol Address)==정보를 주고 받을 수 있는 (=전화번호라고 생각하면된다.)
IP 주소는 기억하기 어렵다-> 도메인 네임으로 대체
DNS(Domain Name System)
(접속하는 순서) == 내컴퓨터 ->도메인 네임 서버->해당 홈페이지 
도메인을 사용하고 싶으면 = ( http://opentutorials.org/course/228/1450 )에서 배워라.

*(1-2)IPv6
자신의 IP를 알아라 -> 접속성공(공인IP사용)/접속실패(사설IP사용)
IP=0.0.0.0 ~ 255.255.255.255 ==42억개의 IP 만들 수 있다.
IPv4는 주소고갈로 IPv6가 나왔다. IPv6는 2의128승개를 만들수 있다.

*(1-3)포트
공유기란? (공유기도 IP를 가지고 있다.=공인IP)
사설IP == 내부망 IP를 얘기한다.
포트(port=항구)
포트포워딩 == 포트를 다른사람에게 그대로 전달할때 사용한다. 공유기가 사설IP로 포워딩하는 걸 의미.

*(1-4)포트포워딩
통신사와 계약한 케이블 = WAN (공인IP 부여)
서버가 설치되어있는 컴퓨터 = LAN1 ,LAN2, LAN3, ....(사설IP)
192.168.0.1 입력하고 들어가면 공유기 관리자화면으로 들어가짐
공유기한테 포트포워딩을 설정한다. 웹서버로 사용할 사설IP의 IP주소를 알아야한다.

*(1-5)절대경로, 상대경로 (파일 php/index.php , path/1.html , path/same.html , sub/sub.html )
절대경로 : 서울시 서울동 서울 아파트 302호 (절대적인 위치를 의미함)
나의 위치와는 무관합니다.
상대경로 : 서울시 서울동 서울 아파트 301호에 살고 있는 사람이 말하는 "옆 집"
나의 위치에 따라서 달라집니다.
점을 하나 쓰면 현재 디렉토리를 의미한다.=( ./ )
점을 두개 쓰면 부모 디렉토리를 의미한다.=( ../ )
최상위 디렉토리를 의미한다.=( / )
하나의 사이트 안에서 이미지 로드시 상대경로를 쓴다.
이 사이트와는 상관없는 이미지를 가져오거나 사이트 밖의 이미지를 가져올때는 절대경로를 쓴다.

*(1-6)유동IP와 DDNS
유동IP( dynamic address) ==계속 바뀌는 IP 주소 (통신사가 가지고 있는 IP가 한정적이기 때문에 유동적으로 운영한다.)
DynamicDNS = IP 정보를 계속 불러온다.(도메인과 호스트의 IP를 지속적으로 일치화(=동기화)시킨다.)
무료로 사용할 수 있는 DDNS 서비스 ( http://freedns.afraid.org/ )
