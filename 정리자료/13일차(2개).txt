*(1-1)호스팅과 클라우드 -웹호스팅VS서버호스팅
HOST :인터넷에 접속된 컴퓨터를 의미한다.
호스팅 : 서비스들을 제공하는 사업자를 의미
IDC(Internet Data Center) : 컴퓨터의 집, 서버가 동작하기에 최적의 조건이 있는 곳.
웹호스팅 : 웹 애플리케이션 운영을 위한 모든 것을 제공.
(애플리케이션을 업로드만 하면됨 but 웹 애플리케이션을 위한 용도로만 사용)
서버호스팅 : 컴퓨터를 제공.
(모든 것을 할 수 있다. but 직접 운영해야 한다.)

*(1-2)클라우드 컴퓨팅
가상화, 종량제(쓰는만큼 돈을 낸다)
가상머신이란? 가짜로 만든 컴퓨터=물리적인 컴퓨터는 아닌데 물리적인 컴퓨터처럼 작용한다.
클라우드컴퓨팅은 컴퓨터의 필요한 양을 조절 가능하다.
(가상화시스템 활용, 필요한 컴퓨터를 유동적으로 만들고 삭제하면서 컴퓨팅 자원 관리)
직접운영시(컴퓨터,운영체제,platform,software 를 직접 설치 후 사용)
클라우드컴퓨팅사용시( IaaS(Infrastructure as a Service), PaaS(Platform as a Service), SaaS(Software as a Service)
IaaS =서버호스팅 해당= 컴퓨터,운영체제 제공 / 나머지 Platform,Software 는 직접 설치
PaaS =웹호스팅 해당= 컴퓨터,운영체제,Platform 제공 / Software 는 직접 설치
SaaS =보통사이트들 해당= 컴퓨터,운영체제,Platform,Software 전부 제공

*(2-1)성장 -성장1
하나의 컴퓨터 안에서 동작함.
Scale UP : 한대의 컴퓨터를 업그레이드 하는 방법 ex)제너럴셔먼나무
Scale OUT : 적당한 크기의 여러 대의 컴퓨터가 작업을 처리하는 방법 ex)판도나무
인프라 : 플랫폼이 설치된 한대의 컴퓨터를 의미
